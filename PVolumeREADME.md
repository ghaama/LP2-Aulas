## Repositório dedicado as aulas de Linguagem de Programação II

### __* Aula 3 (10/03)__

Nesta aula aprendemos fórmulas matemáticas para implantar ao nosso projeto, em que na interface ele pede valores para fazer o cálculo de volume a partir da altura e do raio do objeto.

![alt text](<PS PVolume.rar.jpg>)

Sendo duas caixas de texto para colocar valores e uma caixa de texto trancada para apenas mostrar o valor da fórmula.

![alt text](<PS CODE PVolume.rar.jpg>)

Para fazer o cálculo, aprendemos as fórmulas ___Math.PI___ e ___Math.Pow___, em que na primeira se atribui contas com o valor de Pi e a segunda para adicionar alguma potência à fórmula.
