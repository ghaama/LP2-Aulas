## Repositório dedicado as aulas de Linguagem de Programação II

### __* Aula 2 (03/03)__

Na segunda aula que tivemos, criamos um programa que armazena e envia dados de empregados de uma empresa, porém aprofundando cada vez mais.

Renomeamos cada objeto na interface, usando abreviações para eles, como Label sendo lbl, TextBox como txt e Button para btn. Isso consegue facilitar e muito para nomear cada objeto, onde podemos saber mais rapidamente, direto no código, qual é cada programação e saber em que ela está inserida.


![alt text](<PS PTestes.rar.jpg>)


Na parte de código, é importante também, quando um funcionário não coloque um dado em alguma TextBox, uma mensagem seja enviada e volte a "focar" na parte em que falta.

No código, esse "focar" se atua em uma função chamada "Focus()", voltando para a caixa vazia quando sobresair o erro.
